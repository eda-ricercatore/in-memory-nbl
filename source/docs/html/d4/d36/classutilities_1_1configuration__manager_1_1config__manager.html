Method to set the location of simulation/experimental results.

Parameters
location- Location of a directory.
Returns
the absolute path of location, if location is a relative path from the user's home directory. Else, return location.

IMPORTANT NOTES: Method to fix bug in the method

"os.path.expanduser", which had replaced the initial substring "~/" of the relative path from the user's home directory with "../" This bug is not always replicable. This is because os.environ['HOME'] is set to "../". Hence, it will replace "~/" with "../". Hence, I am providing/using this workaround. O(1) method.
